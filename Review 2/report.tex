\section{Problem Formulation \& Research Gap}

\textbf{Healthcare Context:} In Intensive Care Units (ICUs), continuous monitoring of patient vitals (Heart Rate, Blood Pressure, Oxygen Saturation) generates high-frequency multivariate time-series data. Predicting life-threatening risks (e.g., Sepsis, Cardiac Arrest) is critical for early intervention.

\textbf{The Challenges:}
\begin{enumerate}[noitemsep, leftmargin=1.5em]
    \item \textbf{Privacy Constraints:} Legal frameworks (HIPAA, GDPR) prohibit sharing raw patient data between hospitals.
    \item \textbf{Patient Heterogeneity:} Differences in patient demographics, medical history, and hospital protocols create non-IID data distributions.
    \item \textbf{Internal Concept Drift:} A patient’s physiological state shifts over time (deterioration vs. recovery), rendering static models obsolete.
    \item \textbf{Explainability Need:} Clinicians require transparent reasoning (XAI) to trust AI-driven risk alerts.
\end{enumerate}

\textbf{Research Gap:} Existing Federated Learning (FL) methods like \textit{FedAvg} lack personalization and drift awareness, while personalized methods like \textit{FedPer} do not provide explainability or monitor for active behavioral shifts. Our proposed \textbf{FPDAF} (Federated Personalized Drift-Aware Attention Framework) bridges this gap by integrating all four components into a unified healthcare analytics pipeline.

\section{Technical Validation \& Methodology}

\textbf{Proposed Architecture (FPDAF):} The framework utilizes a split-layer Federated Learning approach. A \textbf{Global Temporal Backbone} (LSTM/Transformer) captures general medical patterns across hospitals, while local \textbf{Personalization Heads} and \textbf{Temporal Attention Layers} adapt to specific patient distributions.

\begin{figure}[H]
\centering
\includegraphics[width=0.65\linewidth]{healthcare_fl_architecture.png}
\caption{System Architecture showing Federated Aggregation with Local Personalization and Attention Mechanism at each hospital node.}
\end{figure}

\textbf{Methodological Novelty:}
\begin{itemize}[leftmargin=1.5em]
    \item \textbf{CUSUM-based Drift Detection:} We monitor the cumulative sum of prediction residuals $S_t = \max(0, S_{t-1} + (e_t - \mu - k))$. When $S_t$ exceeds a threshold $H$, the local personalization head is reset and retrained on recent data, ensuring the model tracks the latest patient state.
    \item \textbf{Attention-Driven Explainability:} The attention mechanism computes weights $\alpha_t$ for historical vitals, allowing clinicians to see which specific physiological event (e.g., a sudden BP drop at T-2 hours) triggered a risk prediction.
\end{itemize}
